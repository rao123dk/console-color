# console-color

# install 
```sh
npm i @rao123dk/console-color

yarn add @rao123dk/console-color
```

# setup
require it in entry file of project.
```sh
require(`@rao123dk/console-color`)
```


# use

```javascript
console.log.red('red color');
```

```javascript
console.log.green('green color');
```
